Proyecto Final de Sistemas Distribuidos

Nuestro proyecto consta por una base de datos creada en Heidi usando mysql, una clase Servidor y una clase Cliente.

Primero se debera ejecutar el servidor para iniciarlo.
Segundo sin cerrar el servidor se ejecuta el Cliente, una vez se ejecute saldra el menu principal de registro e inicio de sesion.

Cabe recalcar que el proyecto hasta ahora fue implementado de manera local y no son funcionales el 100% de  los comandos, solo algunos.
